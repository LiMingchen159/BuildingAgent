# BMS 本地数据库 API — 前端接入说明（更新）

## 1. 基址（重要）

```
http://117.72.185.234/bms
```

**不要用** `117.72.185.234:8765`（公网 8765 未放行）。

所有路径在 `/bms` 后面，例如 `/bms/health`、`/bms/api/v1/points`。

## 2. 数据范围（已扩充）

| 设备 | 合同 trend 起始 (HKT) | 约点数 |
|------|----------------------|--------|
| WCC High Level | 9-Apr-26 | 473 |
| 干接点 / CHP / 冰场 | 29-Apr-26 | ~1,600 |
| 海水泵 SWP | 12-May-26 | 185 |
| **合计** | | **2,261** |

历史已从合同日起回填到库中；之后每 **15 分钟** 自动增量更新。

## 3. 接口就两类（不用区分「历史」和「轮询」）

历史曲线和近期轮询都在同一张表、同一接口里，**不用传 `source`**（默认就是 poll，含 vault 回填的历史）。

### 查历史 / 画曲线

```
GET /bms/api/v1/readings?name={点位名}&from={ISO时间}&to={ISO时间}&limit=5000
```

示例：

```
GET /bms/api/v1/readings?name=WCC_4_SUWT&from=2026-04-09T00:00:00&to=2026-06-12T23:59:59&limit=50000
```

返回：

```json
{
  "total": 3200,
  "items": [
    {
      "point_id": 80,
      "name": "WCC_4_SUWT",
      "ts": "2026-06-12T01:51:15+00:00",
      "value_num": 12.21,
      "value_text": "12.2138"
    }
  ]
}
```

- **时间字段**：`ts`（UTC，带时区）
- **数值**：优先用 `value_num`
- 默认按 `ts` **降序**；画曲线请在前端 **反转为升序**

### 查当前值 / 点位列表

```
GET /bms/api/v1/points?q=WCC_4
GET /bms/api/v1/points?q=WCC_4_SUWT&limit=5
```

看 `last_value`、`last_polled_at`（约 15 分钟延迟，可接受作「近实时」）。

### 健康检查

```
GET /bms/health
→ {"status":"ok"}
```

## 4. 和旧文档的差异（请按这个来）

| 旧写法 | 现在 |
|--------|------|
| `:8765` 直连 | 改用 `/bms` 反代 |
| `/api/v1/timeseries` | **不存在**，用 `/api/v1/readings` |
| 只有 472 个 WCC 点 | **2,261** 点（含 CHP、SWP、压力、COP 等） |
| 要区分 `source=poll` / `history` | **不用管**，省略 `source` 即可 |
| 浏览器直连 enteliWEB `:20800` | **不要**，会 CORS / 鉴权失败 |

## 5. 前端示例（JavaScript）

```javascript
const BMS = 'http://117.72.185.234/bms';

// 历史曲线（一条接口搞定）
async function getHistory(name, from, to) {
  const params = new URLSearchParams({ name, from, to, limit: '50000' });
  const res = await fetch(`${BMS}/api/v1/readings?${params}`);
  const { items } = await res.json();
  return items.reverse(); // 转成时间升序再画图
}

// 当前值
async function getLatest(name) {
  const res = await fetch(`${BMS}/api/v1/points?q=${encodeURIComponent(name)}&limit=1`);
  const { items } = await res.json();
  return items[0]?.last_value;
}
```

## 6. 常用点位名（举例）

| 用途 | 点位名 |
|------|--------|
| WCC_4 出水温度 | `WCC_4_SUWT` |
| WCC_4 电机功率 | `WCC_4_TLKW` |
| WCC_4 COP | `WCC-L1-04_COP`（在干接点表，非 HL） |
| CHP 电压 | `1F_300A_CHP_1P_D07_V_L1` |
| 海水泵 | `SWP_01_V_L1` |

不确定名字时：`GET /bms/api/v1/points?q=关键词&limit=20`

## 7. 实时毫秒级数据

本地库 **不提供** 毫秒级实时；要直连 enteliWEB 需后端代理，**不要在前端写账号密码**。
